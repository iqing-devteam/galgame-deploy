# 后端接口

> 登录/注册接口

```
Method GET

http://bilibili.iqing.in/bilibili_login/?test=1&uname=1_15033405428099267212&token=1_150340542809976434
```

uname: 用户昵称

token:  合作方token，类似于id

sign: 前面query的签名验证

test：当test=1的时候跳过sign验证\(方便测试\)
